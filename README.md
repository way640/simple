# 《幸存者：卡牌进化》Cocos Creator项目

## 项目信息
- **游戏名称**: 幸存者：卡牌进化
- **类型**: Roguelite动作游戏
- **平台**: 微信小游戏
- **引擎**: Cocos Creator 3.x

## 开发目标
实现微信小程序Roguelite游戏《幸存者：卡牌进化》，核心玩法为自动战斗、获取卡牌、装备养成，核心乐趣为发现"机制Bug"快感。

## 核心功能
1. 卡牌系统 - 120张卡牌（基础数值卡60张，角色专属卡20张，无界机制卡20张，终极金卡20张）
2. 容量系统 - 玩家可装备30-60点容量的卡牌
3. 角色系统 - 4个角色（雷恩、薇拉、卡尔、艾莉）
4. 战斗系统 - 自动战斗，怪物生成，伤害计算
5. 经济系统 - 金币
