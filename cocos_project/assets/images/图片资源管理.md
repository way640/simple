# 图片资源管理

## 图片资源管理规范

### 1. 图片命名规范
```
- 角色图片: 角色名称.png
- 怪物图片: 怪物名称.png
- 卡牌图片: 卡牌名称.png
- Boss图片: Boss名称.png
- 特效图片: 特效名称.png
- UI图片: UI名称.png
- Sprite Sheet图片: Sprite Sheet名称.png
- 动画图片: 动画名称.png
```

### 2. 图片尺寸规范
```
- 角色图片: 128x128
- 怪物图片: 64x64
- 卡牌图片: 128x256
- Boss图片: 256x256
- 特效图片: 64x64
- UI图片: 按实际需求
- Sprite Sheet图片: 2048x2048
- 动画图片: 按实际需求
```

### 3. 图片格式规范
```
- 所有图片使用PNG格式
- PNG支持透明度和高质量
- 压缩质量使用"高"或"中"
- 颜色深度使用24位
- 透明度必须有
```

### 4. 图片大小规范
```
- 角色图片: 不超过100KB
- 怪物图片: 不超过50KB
- 卡牌图片: 不超过80KB
- Boss图片: 不超过150KB
- 特效图片: 不超过40KB
- UI图片: 不超过200KB
- Sprite Sheet图片: 不超过2MB
- 动画图片: 不超过100KB
```

```
1. 使用Sprite Sheet合并图片
2. 使用PNG格式，优化图片质量
3. 使用压缩工具压缩图片
4. 删除无用图片
5. 使用懒加载技术
6. 使用缓存技术
7. 使用异步加载
8. 使用预加载技术
```
