# 《幸存者：卡牌进化》Cocos Creator项目

## 项目信息
- **游戏名称**: 幸存者：卡牌进化
- **类型**: Roguelite动作游戏
- **平台**: 微信小游戏
- **引擎**: Cocos Creator 3.x

## 开发目标
实现微信小程序Roguelite游戏《幸存者：卡牌进化》，核心玩法为自动战斗、获取卡牌、装备养成，核心乐趣为发现"机制Bug"快感。

## 核心功能
1. 卡牌系统 - 120张卡牌（基础数值卡60张，角色专属卡20张，无界机制卡20张，终极金卡20张）
2. 容量系统 - 玩家可装备30-60点容量的卡牌
3. 角色系统 - 4个角色（雷恩、薇拉、卡尔、艾莉）
4. 战斗系统 - 自动战斗，怪物生成，伤害计算
5. 经济系统 - 金币、钻石、卡牌碎片
6. 关卡系统 - 5个章节，无尽模式

## 开发计划
### MVP阶段（1-2周）
1. 基础战斗系统（角色移动、怪物生成、伤害计算）
2. 核心数值系统（伤害公式、角色数值、怪物数值）
3. 简单UI（战斗界面、角色状态、怪物状态）

### Alpha阶段（3-4周）
1. 卡牌系统（卡牌拾取、升级、容量）
2. 角色技能系统（技能实现、冷却、伤害）
3. 怪物系统（Boss实现、技能、掉落）

### Beta阶段（5-6周）
1. 经济系统（金币获取、卡牌升级消耗、商城）
2. UI完善（完整UI、设置界面、商城界面）
3. 测试系统（数值验证、Build测试、Boss测试）

## 技术架构
### 目录结构
```
/assets
  /scripts - TypeScript脚本
  /scenes - 场景文件
  /prefabs - 预制体
  /animations - 动画
  /sounds - 音频
  /fonts - 字体
/data
  /configs - 配置文件
  /json - JSON数据文件
/public
  /images - 图片资源
  /icons - 图标资源
```

### 核心脚本
1. `CharacterController.ts` - 角色控制器
2. `DamageCalculator.ts` - 伤害计算器
3. `CardSystem.ts` - 卡牌系统
4. `MonsterGenerator.ts` - 怪物生成器
5. `EconomySystem.ts` - 经济系统

## 微信小游戏适配
- 图片文件大小小于100KB
- 音频文件大小小于200KB
- 游戏总包大小小于10MB
- 帧率不低于30FPS
- 加载时间不超过5秒
