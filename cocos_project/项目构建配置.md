### **微信小游戏适配**

#### **1. 微信API集成**
```
import { WeChatAdapter } from './WeChatAdapter';

const adapter = new WeChatAdapter();

// 微信登录
adapter.login().then(res => {
  console.log('微信登录成功:', res);
});

// 获取用户信息
adapter.getUserInfo().then(res => {
  console.log('获取用户信息成功:', res);
});

// 存储游戏数据
adapter.setStorage('game_data', { score: 100 }).then(() => {
  console.log('存储数据成功');
});

// 读取游戏数据
adapter.getStorage('game_data').then(data => {
  console.log('读取数据成功:', data);
});

// 分享游戏
adapter.shareGame('《幸存者：卡牌进化》', 'share_image.png').then(() => {
  console.log('分享游戏成功');
});

// 支付
adapter.requestPayment(6).then(() => {
  console.log('支付成功');
});

// 性能优化
adapter.optimizeForMemory();
adapter.optimizeForLoadingTime();
adapter.optimizeForCompatibility();
adapter.handleWeChatLimitations();
```

```
微信小游戏适配.md

# 微信小游戏适配指南

## 1. 微信API适配
- wx.login - 微信登录
- wx.getSetting - 用户授权
- wx.getUserInfo - 获取用户信息
- wx.setStorage - 数据存储
- wx.getStorage - 数据读取
- wx.shareAppMessage - 分享游戏
- wx.requestPayment - 支付
- wx.getPerformance - 性能信息
- wx.getDeviceInfo - 设备信息
- wx.getNetworkType - 网络状态

## 2. 微信限制处理
- eval限制 - 不能使用eval
- Function限制 - 不能使用Function
- global限制 - 不能使用global
- 包大小限制 - 不超过10MB
- 性能限制 - 帧率不低于30FPS，内存不超过200MB，加载时间不超过5秒
- 发热限制 - 发热不能导致设备过热

## 3. 微信优化策略
- 图片优化 - 最大尺寸不超过100KB，建议尺寸不超过50KB，使用PNG或JPEG格式
- 音频优化 - 最大尺寸不超过200KB，建议尺寸不超过100KB，使用MP3或WAV格式
- 代码优化 - 总包大小不超过10MB，使用微信小游戏打包工具压缩，删除无用代码
- Sprite Sheet优化 - 角色动画合并到Sprite Sheet中，怪物动画合并到Sprite Sheet中，UI图标合并到Sprite Sheet中，Sprite Sheet尺寸2048x2048，Sprite Sheet数量不超过10个
- 性能优化 - 帧率不低于30FPS，内存不超过200MB，加载时间不超过5秒，发热不会导致设备过热
- 加载时间优化 - 首次加载不超过5秒，使用懒加载技术，延迟加载非必要资源
- 场景切换优化 - 场景切换不超过2秒，使用预加载技术，缓存常用资源
- 资源加载优化 - 资源加载不超过3秒，使用压缩技术，使用异步加载
- 网络优化 - 网络延迟不超过500ms，带宽消耗不超过1MB/s，网络稳定性不低于95%
- 兼容性优化 - iOS设备兼容性，Android设备兼容性，微信版本兼容性，系统版本兼容性，分辨率兼容性

## 4. 微信功能要求
- 分享功能 - 必须有分享功能
- 登录功能 - 必须有登录功能
- 支付功能 - 必须有支付功能
```
